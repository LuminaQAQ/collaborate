# 云迹

## 简介

云迹是一个基于云的文档管理平台，用户可以创建、编辑、分享文档，想法来自[语雀](https://www.yuque.com/)。

主要的目的就是利用这个项目来对所学的知识进行一个实战，或者说总结。也借此学习一下可能会遇到的新的知识。
